# make

多语言开发工具、构建命令和项目模板速查仓库。

## 内容

- `lang.yaml`：常见语言的构建工具、包管理器、项目生成器、编译工具和安装方式清单。
- `android/`、`java/`、`js/`、`python/`、`c++/` 等目录：按语言或平台整理的常用命令。
- `c++/make/`：Makefile 入门示例和可复用模板。
- `c++/cmake/`：CMake 项目模板片段。

## 使用方式

这些 `.sh` 文件主要是命令参考和轻量脚本，不是统一的一键安装入口。运行前请先打开确认内容，尤其是全局安装、系统目录部署、SDK 安装和后台进程相关命令。

可以运行基础校验，检查脚本语法和示例工程：

```sh
./scripts/check.sh
```

## 约定

- 命令示例尽量保持幂等、可读，并避免隐式执行高风险操作。
- 涉及全局安装、系统目录、移动端 SDK 或后台进程的命令，需要先确认本机环境。
- 新增语言或工具时，优先同步更新 `lang.yaml` 和对应目录的命令参考。
